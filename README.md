
### How to run

1) Clone the repo

2) Navigate to cloned directory

3) npm install

4) npm start

Application will run on [http://localhost:3000](http://localhost:3000)

Application hosted on [https://vivriti-6b67e.firebaseapp.com/](https://vivriti-6b67e.firebaseapp.com/)

### To run test cases

1) npm run test
